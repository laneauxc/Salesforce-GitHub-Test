# GitHub Pages Site Preview

## Expected Site Structure

When the GitHub Pages site is live at `https://laneauxc.github.io/Salesforce-GitHub-Test/`, visitors will see:

### Homepage
- **Header**: Site title and tagline with gradient background
- **Navigation Menu**: Home | Features | Bug Fixes | Support | View on GitHub
- **Content**: 
  - Welcome message
  - Overview of documentation sections
  - Quick links to all documentation pages
  - Repository link in footer

### Features Section (`/features/`)
- List of all features with descriptions
- Links to detailed feature documentation:
  - Order Tracking API Integration
  - User Permission Audit Logging

### Bug Fixes Section (`/bugfixes/`)
- List of all bug fixes and updates
- Links to detailed fix documentation:
  - Invoice Calculation Fix
  - Security Patch - Authentication Vulnerability
  - Reporting Dashboard Fixes

### Support Section (`/support/`)
- Support resources and documentation
- Links to support materials:
  - Support Documentation Updates

## Visual Design

The site uses the **Cayman theme** with custom enhancements:

- **Color Scheme**: 
  - Primary: Green gradient header (#159957)
  - Links: Blue (#1e6bb8)
  - Background: Clean white with subtle gray accents

- **Navigation**: 
  - Horizontal menu in header
  - Hover effects for better UX
  - Responsive design for mobile devices

- **Typography**:
  - Clear headings with visual hierarchy
  - Readable font sizes
  - Proper spacing for content sections

- **Layout**:
  - Centered content (max-width: 64rem)
  - Consistent padding and margins
  - Breadcrumb navigation on individual pages

## Responsive Design

The site is fully responsive:
- **Desktop**: Full navigation menu, wide content area
- **Tablet**: Adjusted navigation, optimized content width
- **Mobile**: Stacked navigation menu, narrow content area

## Accessibility

- Semantic HTML structure
- Proper heading hierarchy
- Keyboard navigation support
- ARIA labels where appropriate
